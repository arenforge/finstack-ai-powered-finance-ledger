# Client

This folder contains the Vite + React frontend for Personal Finance Ledger.

## Scripts

- `npm run dev` starts the local Vite dev server on `http://localhost:5173`
- `npm run start` aliases the dev server
- `npm run build` creates a production build in `dist/`
- `npm run preview` previews the production build locally
- `npm test` runs the Vitest test suite

## Environment

Copy [client/.env.example](</D:/1. New Study Material/Personal Finance Ledger/client/.env.example>) to `client/.env` and fill in the Firebase and API values.

The frontend still uses `REACT_APP_` env keys, and the Vite config is set up to read them.
